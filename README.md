# Makers IA Marketplace

Community-curated collection of Claude Code plugins, created and shared by [Makers IA](https://makersia.com) members.

## Install

```bash
claude plugin add mia-marketplace
```

## Plugins

| Plugin | Description |
|--------|-------------|
| [mia-statusline](https://github.com/crystian/crystools-statusline) | Status line with context window, git, cost, rate limits, and cache stats. |

## Contributing

Members of the Makers IA community can submit plugins to this marketplace. Each plugin must follow the [Claude Code plugin structure](https://docs.anthropic.com/en/docs/claude-code/plugins).
