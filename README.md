# Makers IA Marketplace

Community-curated collection of Claude Code plugins, created and shared by [Makers IA](https://makersia.com) members.

## Install

```bash
claude plugin marketplace add crystian/mia-marketplace
```

## Plugins

| Plugin | Description |
|--------|-------------|
| [crystools-statusline](https://github.com/crystian/crystools-statusline) | Status line with context window, git, cost, rate limits, and cache stats. |
| [crystools-agents-talk](https://github.com/crystian/crystools-agents-talk) | Chat-like monitor for agent-to-agent communication. Logs colored, timestamped messages with duration, tokens, and tool counts. |
| [crystools-skills](https://github.com/crystian/skills) | Open collection of AI agent skills — reusable, framework-agnostic SKILL.md packages. |

## Contributing

Members of the Makers IA community can submit plugins to this marketplace. Each plugin must follow the [Claude Code plugin structure](https://docs.anthropic.com/en/docs/claude-code/plugins).
